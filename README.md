Active Regulation & Control of Thermal Interfaces for Cryogenic Fridge Operation eXperiments (ARCTIC FOX)
Reads thermometers and controls various heaters for CTC100, LakeShore224 and Lakeshore372 devices
Consists of two main scripts;
controls.py for the widgets that can controll heaters, heat switches and still heaters
temperature_monitor.py for live plotting and writing to h5 files
